Privacy Policy

Last updated: 16 February 2026

Overview

UsageClaw is a browser extension and web dashboard that helps you monitor AI API usage across providers. We are committed to protecting your privacy.

Data Collection

We do not collect, transmit, or store any user data on our servers.

  • API keys you enter are stored locally in your browser (localStorage or chrome.storage.local) and are never sent to UsageClaw or any third party.
  • Usage data is fetched directly from each AI provider's API using your own keys and displayed locally.
  • No analytics, tracking, or telemetry is included.

Data Storage

  • Chrome Extension: API keys and cached usage data are stored in chrome.storage.local, which is sandboxed to the extension.
  • Web App: API keys are stored in your browser's localStorage.
  • You can delete all stored data at any time by clearing the extension's storage or your browser data.

Third-Party Services

The extension connects directly to the following APIs only when you provide your own API key:

  • OpenAI (api.openai.com)
  • DeepSeek (api.deepseek.com)
  • MiniMax (minimax.io)

Each provider's own privacy policy applies to data exchanged with their APIs.

Permissions

  • storage: To save API keys and usage data locally.
  • alarms: To schedule periodic usage data refresh.
  • Host permissions: To make direct API calls to AI providers listed above.

Children's Privacy

UsageClaw is not directed at children under 13. We do not knowingly collect data from children.

Changes

We may update this policy from time to time. Changes will be posted on this page.
